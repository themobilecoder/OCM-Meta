# OCM-Meta
Go server for to provide a list of OnChainMonkey metas from [Metagood OCM Repo](https://github.com/metagood/OnChainMonkeyData)


## Installation
### Mac

Using Homebrew

```bash
brew install go-task/tap/go-task
```


## Usage

```bash
$ task build
$ bin/ocm_meta
```
